# Task Manager

A simple PHP Task Manager application with:

- User registration
- User login
- Token-based authentication
- Create task
- View tasks with pagination
- Filter tasks by status
- Update task
- Soft delete task
- Plain HTML, CSS, and JavaScript frontend

## Technologies
- PHP
- MySQL
- PDO
- JavaScript
- HTML/CSS
- Composer
- phpdotenv
